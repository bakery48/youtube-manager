# 📺 YouTube Channel Manager

YouTubeの登録チャンネルをフォルダで管理できるWebアプリです。

## ✨ 機能

- ✅ **Googleアカウント連携**: OAuth 2.0で自動ログイン
- ✅ **自動チャンネル取得**: 登録チャンネルを自動で同期
- ✅ **フォルダ管理**: チャンネルをカテゴリごとにフォルダ分け
- ✅ **新着動画一覧**: フォルダごとに新着動画を表示
- ✅ **未視聴/視聴済み管理**: 動画の視聴状態を管理
- ✅ **お気に入り**: お気に入り動画を保存
- ✅ **検索機能**: タイトルやチャンネル名で検索
- ✅ **ソート機能**: 日付順、チャンネル順で並び替え
- ✅ **ダークモード**: 目に優しいプレミアムデザイン

## 🚀 セットアップ手順

### 1. Google Cloud Consoleでプロジェクトを作成

1. [Google Cloud Console](https://console.cloud.google.com/) にアクセス
2. 新しいプロジェクトを作成
   - プロジェクト名: `youtube-manager` (任意)

### 2. YouTube Data API v3を有効化

1. 左メニュー → 「APIとサービス」 → 「ライブラリ」
2. 検索バーで「YouTube Data API v3」を検索
3. クリックして「有効にする」

### 3. OAuth 2.0 クライアントIDを作成

1. 左メニュー → 「APIとサービス」 → 「認証情報」
2. 「認証情報を作成」 → 「OAuth クライアント ID」
3. 同意画面の設定(初回のみ):
   - ユーザータイプ: 「外部」を選択
   - アプリ名: `YouTube Manager` (任意)
   - ユーザーサポートメール: 自分のメールアドレス
   - デベロッパーの連絡先情報: 自分のメールアドレス
   - 「保存して次へ」
   - スコープは設定不要、「保存して次へ」
   - テストユーザーに自分のGoogleアカウントを追加
4. OAuth クライアント IDの作成:
   - アプリケーションの種類: 「ウェブアプリケーション」
   - 名前: `YouTube Manager Web Client` (任意)
   - 承認済みのJavaScript生成元: 
     - ローカルで使う場合: `http://localhost` または `file://`
     - ※ファイルから直接開く場合は設定不要
   - 「作成」をクリック
5. 表示されたクライアントIDをコピー
   - 形式: `xxxxxxxxxx.apps.googleusercontent.com`

### 4. APIキーを作成

1. 「認証情報を作成」 → 「APIキー」
2. 作成されたAPIキーをコピー
   - 形式: `AIzaSy...`

### 5. アプリを起動

1. `index.html` をブラウザで開く
2. 右上の⚙️ボタンをクリック
3. 設定画面で以下を入力:
   - **クライアントID**: 手順3でコピーしたもの
   - **YouTube Data API キー**: 手順4でコピーしたもの
   - **取得する動画数**: 10 (デフォルト)
4. 「保存」をクリック

### 6. Googleアカウントでログイン

1. 右上の「Googleでログイン」ボタンをクリック
2. Googleアカウントを選択
3. アクセス許可を承認
4. 自動的に登録チャンネルが取得されます!

## 📖 使い方

### チャンネルの管理

- **自動取得**: ログインすると自動的に登録チャンネルが追加されます
- **手動追加**: 「チャンネルを追加」ボタンから個別に追加も可能
- **フォルダ分け**: チャンネルをドラッグ&ドロップでフォルダに移動(予定)

### フォルダの作成

1. サイドバーの➕ボタンをクリック
2. フォルダ名とカラーを設定
3. 「保存」をクリック

### 動画の視聴

- **動画カードをクリック**: YouTubeで動画を開く
- **⭐ボタン**: お気に入りに追加/解除
- **✅ボタン**: 視聴済みにマーク

### 動画の更新

- 「🔄 更新」ボタンをクリックすると、全チャンネルの新着動画を取得します

## 📁 データ保存

すべてのデータはブラウザのLocalStorageに保存されます:
- フォルダ設定
- チャンネル情報
- 動画データ
- 視聴済み状態
- お気に入り
- ログイン情報(アクセストークン)

## 🎨 カスタマイズ

`style.css` の `:root` セクションでカラーテーマをカスタマイズできます。

```css
:root {
    --accent-primary: #6366f1; /* メインカラー */
    --bg-primary: #0f0f1a;     /* 背景色 */
    /* ... */
}
```

## ⚠️ 注意事項

- **APIクォータ制限**: YouTube Data API には1日あたり10,000ユニットの制限があります
  - チャンネル検索: 100ユニット
  - 動画検索: 100ユニット
  - 登録チャンネル取得: 1ユニット
- **アクセストークンの有効期限**: 約1時間で期限切れになります(再ログインが必要)
- **ブラウザのデータ**: LocalStorageをクリアするとデータが消えます

## 🔧 トラブルシューティング

### ログインできない
- クライアントIDが正しく設定されているか確認
- Google Cloud Consoleで同意画面の設定が完了しているか確認
- テストユーザーに自分のアカウントが追加されているか確認

### チャンネルが取得できない
- ログインしているか確認
- APIキーが正しく設定されているか確認
- APIのクォータが残っているか確認

### 動画が表示されない
- 「🔄 更新」ボタンをクリックして動画を取得
- ブラウザのコンソール(F12)でエラーを確認

## 🔐 セキュリティ

- アクセストークンはLocalStorageに保存されます
- 本番環境で使用する場合は、OAuth同意画面を「公開」に変更してください
- 個人利用の場合は「テスト」モードのままで問題ありません
